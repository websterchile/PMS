Help & Documentation

Getting Started
  • Dashboard: View overall pump health, alerts, and quick stats.
  • Sensor Data: Click on a sensor card to view detailed readings and trends for Temperature and Vibration sensors.
  • Predictions: See the latest predictions for pump maintenance and sensor health.
  • Maintenance: Track when the pump is due for maintenance, see system alerts, and view sensor connection status.
  • Logs: Add, edit, or delete maintenance logs for the pump.
  • Data Management: Upload new sensor data, generate synthetic data, or (soon) connect to live sensors.
Key Features
  • Real-time and historical sensor data visualization
  • Automated maintenance predictions and alerts
  • Maintenance log management (add, edit, delete)
  • CSV data upload and synthetic data generation
  • System health and sensor connection monitoring
  • User-friendly, mobile-responsive interface
